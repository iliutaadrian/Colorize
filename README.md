For this project to work you need to follow:
1. composer update
2. php artisan key:generate
5. npm install

To start the aplication:
1. php artisan serve
2. npm run watch
3. open the http://127.0.0.1:8000

Colorize is a web platform that let's you change the color of your hair.
It is build using:
    - Laravel 5.4
    - Laravel Passport
    - VueJS
